疑似セレクタ（:○○○）

リンク関連の疑似クラス

書式

疑似クラス 説明
:link リンク先をまだ見ていない状態
:visited リンク先を既に見た状態
:hover カーソルが上にある状態
:active マウスのボタン等が押されている状態

CSSコード

疑似クラス コード例
:link a:link{color: blue}
:visited a:visited{color: purple}
:hover a:hover{color: red}
:active a:active{color: yellow}

HTMLコード

疑似クラス コード例
:link <a href="http://www.baidu.com/">リンク先をまだ見ていない状態</a>
:visited <a href="http://www.qq.com/">リンク先を既に見た状態</a>
:hover <a href="http://www.taobao.com/">カーソルが上にある状態</a>
:active <a href="http://www.google.com/">マウスのボタン等が押されている状態</a>
