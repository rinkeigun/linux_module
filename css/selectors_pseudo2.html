疑似セレクタ（:○○○）

疑似クラス

疑似クラス :nth-child(An+B)
説明 先頭からB個目の要素からA個おきに適用
CSS li:nth-child(2n+1){color: red;}
HTML <ol><li>１個目</li><li>2</li><li>３個目、２個おき</li><li>４個目</li></ol>
結果
  1. １個目
  2. 2
  3. ３個目、２個おき
  4. ４個目
疑似クラス :nth-last-child(An+B)
説明 最後からB個目の要素からA個おきに適用
CSS li:nth-last-child(2n+1){color: blue;}
HTML <ul><li>１個目</li><li>後部から２個おき</li><li>３個目、２個おき</li><li>４個目、後部から１個目</li></ul>
結果
  • １個目
  • 後部から２個おき
  • ３個目、２個おき
  • ４個目、後部から１個目
疑似クラス :nth-of-type(An+B)
説明 同じ要素の先頭B個目の要素からA個おきに適用
CSS li:nth-of-type(2n+1){color: yellow;}
HTML <ul><li>１個目</li><li>先頭から２個おき</li><li>３個目、２個おき</li><li>４個目、後部から１個目</li></ul>
結果
  • １個目
  • 先頭から２個おき
  • ３個目、２個おき
  • ４個目、後部から１個目
